# ЕППБ MVP — Единый портал поддержки бизнеса

Публичная спецификация MVP для конкурса АО «НИХ «Байтерек» / Astana Hub.
Проект демонстрирует единый клиентский путь получения мер поддержки и no-code/AI
конструктор новых государственных услуг.

## Публичные контуры

- `https://bizdnai.com/baiterek/take` — подобрать и получить услугу;
- `https://bizdnai.com/baiterek/create` — создать, проверить и опубликовать услугу.

Маршруты зафиксированы как целевые публичные точки входа. Текущий этап репозитория —
утверждённая спецификация и архитектурный каркас; приложение будет развёрнуто на VM.

## Что отличает решение

- Услуги не зашиты в код: каждая описана версионируемым `Service Definition` в JSON.
- Один runtime рендерит формы, правила видимости, расчёты и этапы любой услуги.
- Новую услугу можно получить из текста программы через AI, собрать визуально или импортировать JSON.
- Анкета разбивается на тематические экраны по 3–6 полей и непрерывно сохраняет черновик.
- AI-подбор меры становится началом анкеты: подтверждённые ответы переходят в черновик.
- Клиент продолжает заполнение с другого устройства с сохранённого checkpoint.
- Государственные интеграции в MVP представлены честно обозначенными mock-коннекторами.

## Контрольные услуги

1. «Приобретение авиатранспорта и вагонов в лизинг — I этап» — сложная многоэтапная услуга.
2. «Агробизнес: животноводство» — популярная программа с ветвлениями и расчётами.

Обе услуги должны существовать только как записи `service_definitions`, созданные через
единый конструктор. В прикладном коде не должно быть логики, привязанной к конкретным услугам.

## Быстрый старт

Репозиторий пока находится на стадии `spec-first`: исполняемый код и Docker Compose
будут добавляться по мере реализации. После появления первого вертикального среза
целевой запуск будет выглядеть так:

```bash
cp .env.example .env
docker compose up -d --build
docker compose exec api alembic upgrade head
docker compose exec api python -m app.seed
```

Проверка:

```bash
docker compose ps
docker compose exec api pytest
```
